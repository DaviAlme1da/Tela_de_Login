Projeto desenvolvido na disciplina de Front-End com o objetivo de consolidar os princípios da Programação e aplicar os conhecimentos iniciais em desenvolvimento web.

O sistema consiste em uma página de login funcional, desenvolvida utilizando tecnologias básicas da web.



---

<h2> Abrir e Rodar o Projeto </h2>

- Realize o git clone do projeto.
  
  ```bash
  git clone https://github.com/DaviAlme1da/Tela_de_Login.git
  
---

<h2>🛠️ Conhecimentos Aplicados</h2>

<p>Durante o desenvolvimento deste projeto, pude aplicar e consolidar diversos conhecimentos importantes em Front-End, incluindo:</p>

<ul>
  <li><strong>Controle de versão com Git:</strong> comandos básicos como <code>git init</code>, <code>git add</code> e <code>git commit</code> para gerenciar e versionar o código.</li>
  
  <li><strong>Framework Bootstrap:</strong> aplicação de componentes e layout responsivo para facilitar a construção da interface.</li>
  
  <li><strong>Ferramentas de desenvolvedor do navegador:</strong> aprendizado de como inspecionar elementos e alterar estilos diretamente para testar mudanças rapidamente.</li>
  
  <li><strong>Comentários no código:</strong> utilização de comentários para melhorar a organização e facilitar a compreensão do projeto.</li>
  
  <li><strong>Criação de página de login:</strong> aplicação de formulário de login utilizando recursos do Bootstrap.</li>
  
 <li><strong>Página de Login e suas funcionalidades:</strong> construção de um formulário de login completo utilizando Bootstrap, com campos de <em>e-mail</em> e <em>senha</em>, e validações básicas para simular autenticação de usuário.</li>
  
  <li><strong>Seção FAQ interativa:</strong> criação de uma área de perguntas frequentes para fornecer respostas rápidas aos usuários, melhorando a experiência de navegação..</li>
</ul>
